How to Use This Tool:

  • Prepare an Excel file with columns for Level and Name
  • Click "Import from Excel" to load your organizational data
  • Your Excel file is processed entirely in your browser - no data is sent to any server
  • Use the expand/collapse buttons to view different sections of the hierarchy
  • To use on GitHub Pages: simply upload this HTML file and your Excel file to your repository
  • Your data remains completely private - nothing is stored on GitHub servers
